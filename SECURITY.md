# Security Policy

## Reporting a Vulnerability

If you find content in this repository that appears to enable unsafe
biological reconstruction or misuse, please **open a private security
advisory** on GitHub (Security tab > Report a vulnerability) or contact
the maintainer directly.

We take all reports of biosecurity-relevant content seriously and will
respond promptly.

## Scope

This policy covers concerns about:

- Content that could enable synthesis of infectious material
- Nucleic acid sequences of regulated select agents
- Reverse-genetics or rescue workflows
- Laboratory protocols for pathogen propagation
- Any content that lowers barriers to producing dangerous biological
  material

## What This Repository Is

This repository is for non-infectious computational structure modeling,
coarse-grained relaxation, and visualization. See [SAFETY.md](SAFETY.md)
for the full safety and scope statement.

Ebolaviruses are regulated select agents under U.S. 42 CFR § 73.3.
This repository does not host biological material, infectious nucleic
acids, rescue systems, or laboratory methods.

## Response Timeline

- **Acknowledgment**: within 48 hours
- **Initial assessment**: within 7 days
- **Resolution**: depends on severity; content will be removed or
  corrected as needed
